This plugin adds a widget that displays information on your Final Fantasy XI character.

== Description ==

This information must be manually entered using the provided profile editor. Quick updates to your character (including main job level and a small status field) can be entered using the Dashboard widget.

== Installation ==

1. Upload the 'ffxi-char-profile' folder into the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress administration.
3. Add the widget to your page using the 'Widgets' menu.
